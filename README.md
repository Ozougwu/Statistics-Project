# Statistics Project

## Overview
This repository contains a statistics analysis project built with R and documented in a final report. It includes the project dataset, R Markdown scripts, and the final PDF report used to present methods, results, and conclusions.

## Repository Contents
- `Statistics Project Report.pdf`: Final written report summarizing methods, analysis, and findings.
- `R Scripts.rar`: Archive containing the R Markdown scripts used for modeling and analysis.
- `Group 22 R report.RData`: RData file used as the project dataset/input.

## Features
- Statistical analysis workflow using R Markdown.
- Reproducible scripts for modeling and reporting.
- Final report presentation with conclusions and interpretation of results.
- End-to-end project materials in one repository (data, scripts, and report).

## Getting Started

### Prerequisites
- R (latest stable version recommended)
- RStudio (recommended for running `.Rmd` workflows)
- Required R packages used by the scripts (install as prompted when running)

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Ozougwu/Statistics-Project.git
   cd Statistics-Project
   ```
2. Extract the scripts archive:
   - Unzip/unpack `R Scripts.rar` to access the `.Rmd` files.
3. Ensure the dataset file is available to the scripts:
   - The scripts reference `data_group_22.RData`, while the repository file is `Group 22 R report.RData`.
   - Rename or adjust the load path as needed before running.

## Usage
1. Open the extracted `.Rmd` files in RStudio or your preferred R Markdown environment.
2. Update file paths if required for your local setup.
3. Run the analysis sections in order to reproduce outputs.
4. Review `Statistics Project Report.pdf` for the final summary of results.

## Project Goals
- Demonstrate applied statistical analysis using real project data.
- Provide reproducible scripts that support the final report.
- Present clear findings and interpretations based on the analysis.

## Results
The repository demonstrates a complete statistics project workflow, from dataset loading and scripted analysis to final report delivery with interpretable results.
